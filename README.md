# Dise-o-de-contador-de-12-bits-con-control-de-incremento-decremento-programable
Diseño de un circuito digital secuencial que implementa un contador programable de 12 bits con control de incremento/decremento, carga externa de valores y señalización de estados. Proyecto académico realizado mediante lógica discreta, sin uso de HDL.
